# Neural-Recognition-Project

Este repositório contém o desenvolvimento e implementação manual de uma rede **Multilayer Perceptron (MLP)** e na aplicação de uma **Convolutional Neural Network (CNN)**.

## 1. Multilayer Perceptron (MLP)

Implementação manual da arquitetura **Feedforward** com algoritmo **Backpropagation** (Gradiente Descendente).

* **Dados:** Portas lógicas (AND, OR, XOR) e Conjunto de Caracteres Completo.
* **Tecnologias:** Python e NumPy (para operações de matrizes).

## Estrutura de Arquivos de Saída (MLP)

O código gera:

* `hiperparametros.txt`: Configurações da rede.
* `pesos_iniciais.csv` / `pesos_finais.csv`: Estado das conexões.
* `erro_iteracao.csv`: Histórico do erro por época.
* `saidas_teste.csv`: Resultados das predições.
